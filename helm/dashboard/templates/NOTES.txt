## Next Steps

1. Wait for all pods to be ready:
   kubectl wait --for=condition=ready pod -l app.kubernetes.io/name={{ include "store-platform.name" . }} -n {{ .Release.Namespace }} --timeout=300s

2. Verify ingress is working:
   curl -k https://{{ .Values.api.ingress.host }}/health/ready

3. Create your first store:
   curl -X POST https://{{ .Values.api.ingress.host }}/api/v1/stores \
     -H "Content-Type: application/json" \
     -d '{"name":"my-first-store","engine":"woocommerce"}'

## Useful Commands

# Check pod status
kubectl get pods -n {{ .Release.Namespace }}

# View logs
kubectl logs -n {{ .Release.Namespace }} -l app.kubernetes.io/component=api -f

# List all stores
kubectl get namespaces -l app.kubernetes.io/part-of=store-platform
